## 🎯 Project Overview

### Application Purpose
- **Multi-role Real Estate Platform** for Uttar Pradesh
- Three distinct user roles: Customer, Partner, Admin
- Complete property listing and management system
- Built with React Native & TypeScript

### Technology Stack
- **Frontend**: React Native, TypeScript, Expo
- **Navigation**: React Navigation (Bottom Tabs + Stack)
- **UI/UX**: Custom components with animations
- **State Management**: React Context API
- **Mock Data**: Comprehensive test data for all features

## 🚀 How to Test

### Role Switching

**File**: `src/context/AuthContext.tsx` (Line 33)

```typescript
// Switch between roles:
const CURRENT_ROLE = 'customer';  // Customer Dashboard
const CURRENT_ROLE = 'partner';   // Partner Dashboard
const CURRENT_ROLE = 'admin';     // Admin Dashboard
```

**Steps:**
1. Open `AuthContext.tsx`
2. Change `CURRENT_ROLE` variable
3. Save file
4. Restart the app
5. Test respective dashboard
